Comment utiliser Deskreen

TROIS ÉTAPES FACILES

NOTE
Deskreen partage l'écran de votre ordinateur avec un navigateur Web. Cela signifie N'IMPORTE QUEL un appareil avec un navigateur Web, tel qu'un téléphone, une tablette, un autre ordinateur ou même une télévision intelligente, peut être utilisé pour étendre l'écran de votre ordinateur pendant que vous utilisez Deskreen. AUSSI vous pouvez connecter autant d'appareils que vous le souhaitez ! (téléphones, tablettes, autres ordinateurs, téléviseurs intelligents, etc.)
Watch video demos
1
PREMIÈRE ÉTAPE : CONNECTEZ-VOUS
Lorsque vous avez votre ordinateur avec Deskreen en cours d'exécution - Scannez le code QR avec une tablette ou un téléphone, ou tapez manuellement dans la barre d'adresse du navigateur Web ce que vous voyez dans le bouton bleu sous le code QR. Cela ouvrira une page sur votre tablette avec l'état de la connexion.

Cliquez sur le code QR pour l'agrandir si votre appareil photo ne peut pas reconnaître un petit code QR

Si vous n'avez pas d'appareil photo pour scanner un code QR, cliquez sur le bouton bleu sous le code QR pour copier l'adresse dans le presse-papiers. Ensuite, vous pouvez partager l'adresse copiée et la coller dans la barre d'adresse du navigateur Web pour ouvrir la visionneuse Deskreen. Vous pouvez également taper manuellement l'adresse dans le bouton bleu dans la barre d'adresse du navigateur sur n'importe quel appareil pour ouvrir Deskreen Viewer.

Lorsque vous suivez un lien par code QR ou que vous le saisissez manuellement dans la barre d'adresse du navigateur, vous verrez une boîte de message comme ci-dessous. L'adresse surlignée en vert doit correspondre à ce que vous voyez sur l'application Deskreen de votre ordinateur et sur l'appareil auquel vous essayez de vous connecter (par exemple, téléphone, tablette). Si les adresses correspondent, vous pouvez cliquer sur le bouton Autoriser. Cela vous mènera à la deuxième étape.

2
DEUXIÈME ÉTAPE : SÉLECTIONNER
Lorsque vous avez cliqué sur le bouton "Autoriser", il vous sera demandé de choisir la source d'écran que vous souhaitez partager. Vous serez invité à sélectionner l'une des deux options suivantes : Partager l'écran entier ou Partager la fenêtre de l'application. Comme dans une image ci-dessous.

DEUXIÈME ÉTAPE (a) : Partage de fenêtre d'application

Lorsque vous cliquez sur le bouton de la fenêtre d'application, vous pouvez sélectionner la fenêtre d'application à partager, comme dans l'image ci-dessous. Cliquez sur la boîte de prévisualisation de la fenêtre de l'application et vous passerez à la troisième étape.

ÉTAPE DEUX (b) : Partage d'écran complet (cas d'utilisation d'écran miroir et de deuxième écran)

Si vous souhaitez partager l'intégralité de l'écran, cliquez sur le bouton "Ecran entier". Vous pouvez choisir de partager tout l'écran lorsque vous avez besoin de refléter tout l'écran ou de l'utiliser comme deuxième écran, lorsque l'adaptateur d'affichage virtuel est connecté. Cliquez sur la boîte de prévisualisation et vous passerez à l'étape 3.

3
TROISIÈME ÉTAPE : CONFIRMER
C'est une dernière étape. Ici, vous venez de vérifier, pour vous assurer que tout est comme vous le vouliez. Si vous avez changé d'avis, vous pouvez revenir en arrière ou même déconnecter un appareil en cliquant sur le bouton vert "Connecté", un menu d'informations de connexion s'ouvrira avec le bouton "Déconnecter" sur lequel vous pourrez cliquer pour déconnecter l'appareil.
FINI!
Après tout, vous verrez 'Terminé !' message, et vous pouvez répéter cette procédure pour connecter un autre appareil.

Vous pourrez voir l'écran de votre ordinateur dans un navigateur Web sur un autre appareil. Vous pouvez voir un exemple de partage de fenêtre d'application dans l'image ci-dessous.

Additional Features

Panneau des appareils connectés
En cliquant sur le bouton bleu "Appareils connectés" dans la barre supérieure, le panneau s'ouvrira et vous pourrez déconnecter un ou tous les appareils connectés. En cas d'urgence, quittez simplement l'application Deskreen et toutes les sessions de partage d'écran s'arrêteront immédiatement.

Exigences et autorisations pour l'application Deskreen

Autorisation MacOS pour permettre à Deskreen d'enregistrer l'écran
Deskreen fonctionne via la technologie d'enregistrement d'écran
Permission Required
pour diffuser en direct l'image de l'écran sur votre tablette, votre téléphone. Vous devrez donc fournir une autorisation d'enregistrement d'écran pour Deskreen sur MacOS afin de permettre à Deskreen d'accéder à votre écran.
Autorisation Windows pour permettre à Deskreen d'accéder aux réseaux locaux et publics
Les utilisateurs de Windows doivent ajouter "l'accès au réseau privé" pour Deskreen au premier démarrage ou dans les paramètres du pare-feu
Network Access Required
Les utilisateurs de Windows doivent ajouter "l'accès au réseau privé" pour Deskreen au premier démarrage ou dans les paramètres du pare-feu
Pare-feu, antivirus, etc.
Deskreen n'est pas encore signé. Cela signifie que votre antivirus se plaindra lorsque vous l'exécuterez pour la première fois.
Important Security Notice
Verification costs money. Please donate so we raise funds for making and keeping Deskreen code signed. You should treat Deskreen as an exception for your malware protection software. Also Firewals both on your computer and WiFi router should not be blocking any Deskreen network access and activity. Otherwise Deskreen may not work smooth as expected.
