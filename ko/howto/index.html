화면 공유 및 화면 표시를 위한 Deskreen 사용 방법

간단한 세 가지 단계

노트
Deskreen 화면 공유: 컴퓨터 화면을 웹 브라우저에 공유합니다. 그 의미는 어디에서 전화, 태블릿, 다른 컴퓨터 또는 심지어 스마트 TV와 같은 웹 브라우저가 있는 장치에 화면을 표시하여 Deskreen을 사용하는 동안 컴퓨터 화면을 확장할 수 있습니다. 또한 원하는만큼 많은 장치를 연결할 수 있습니다!(전화, 정제, 다른 컴퓨터, 스마트 TV 등)

1 단계 : 연결
DeskReen 이 있는 컴퓨터가있는 경우 태블릿이나 전화로 QR 코드를 스캔하거나 웹 브라우저 주소 표시 줄에 수동으로 QR 코드 아래에있는 파란색 버튼에서 볼 수 있습니다. 이는 연결 상태의 태블릿에서 페이지를 열 것입니다.

카메라가 작은 QR 코드를 인식 할 수없는 경우 QR 코드를 클릭하십시오.

QR 코드를 스캔 할 카메라가 없는 경우 QR 코드 아래의 파란색 버튼을 클릭하여 주소를 클립 보드에 복사합니다. 그런 다음 복사 된 주소를 공유하고 웹 브라우저 주소 표시 줄에 붙여 넣어 DeskReen Viewer를 열 수 있습니다. 모든 장치의 브라우저 주소 표시 줄에있는 파란색 버튼에 주소를 입력하여 DeskReen Viewer를 열 수 있습니다.

QR 코드로 링크를 따르거나 브라우저 주소 표시 줄에서 수동으로 입력하면 아래와 같은 메시지 상자가 표시됩니다. 녹색으로 강조 표시된 주소는 컴퓨터 데스크레스 앱과 연결하려고하는 장치 (예 : Phone, Tablet)에 표시되는 장치와 일치해야합니다. 주소가 일치하는 경우 허용 버튼을 클릭 할 수 있습니다. 이렇게하면 2 단계로 이어집니다.

2
2 단계 : 선택
'허용'버튼을 클릭하면 공유 할 화면 원본을 선택하라는 메시지가 표시됩니다. 두 가지 옵션 중 하나를 선택하라는 메시지가 표시됩니다. 전체 화면을 공유하거나 응용 프로그램을 공유하십시오. 아래 이미지와 같습니다.

2 단계 (a) : 응용 프로그램 윈도우 공유

응용 프로그램 창 단추를 클릭하면 아래 이미지와 같은 이미지처럼 공유 할 앱 창을 선택할 수 있습니다. 앱 창 미리보기 상자를 클릭하면 3 단계로 진행됩니다.

2 단계 (B) : 전체 화면 공유 (미러 화면 및 두 번째 화면 사용 사례)

전체 화면을 공유하려면 '전체 화면'버튼을 클릭하십시오. 가상 디스플레이 어댑터가 연결되어있을 때 전체 화면을 미러링하거나 두 번째 화면으로 사용해야 할 때 전체 화면을 공유하도록 선택할 수 있습니다. 미리보기 대회상자를 클릭하면 3 단계로 진행됩니다.

3
3 단계 : 확인
이것은 마지막 단계입니다. 여기에서 당신이 원하는대로 모든 것을 확인하기 위해 더블 수표를 두 번 확인합니다. '연결된'녹색 버튼을 클릭하면 다시 연결을 끊거나 장치를 분리 할 수 있습니다. 연결을 끊을 수있는 '연결 끊기' 버튼을 클릭하면 연결 정보 메뉴가 열립니다.
완료!
모든 과정이 끝나면 '완료'가 보일 것입니다. 이 절차를 반복하여 하나의 장치를 연결할 수 있습니다.

다른 장치의 웹 브라우저에서 컴퓨터 화면을 볼 수 있습니다. 아래 이미지에서 응용 프로그램 창 공유의 예제를 볼 수 있습니다.

Additional Features

연결된 장치 패널
상단 막대에서 파란색 '연결된 장치'버튼을 클릭하면 패널이 열리거나 연결된 모든 장치를 분리 할 수 있습니다. 비상 사태가 발생하면 DeskReen 앱을 종료하고 모든 화면 공유 세션이 즉시 중단됩니다.
플립 화면 모드
다음은 플립 화면 모드가 작동하는 방식에 대한 데모입니다. TelePrompter를 사용할 때 유용합니다.

DeskReen 앱의 요구 사항 및 사용 권한

DeskReen이 화면을 기록 할 수있는 MacOS 권한
DeskReen은 화면 녹화 기술을 통해 작동합니다
Permission Required
태블릿에 스트림 화면 그림을 실시간으로 연결합니다. 따라서 DeskReen이 화면에 액세스 할 수 있도록 MacOS에서 DeskReen에 대한 화면 녹음 권한을 제공해야합니다.
DeskReen이 로컬 및 공용 네트워크에 액세스 할 수있는 Windows 권한
Windows 사용자는 처음 시작 또는 방화벽 설정에서 데스크레임에 '개인 네트워크 액세스'를 추가해야합니다.
Network Access Required
Windows 사용자는 처음 시작 또는 방화벽 설정에서 데스크레임에 '개인 네트워크 액세스'를 추가해야합니다.
방화벽, 바이러스 백신 등
Deskreen은 아직 코드 서명되지 않았습니다.즉, 당신이 처음으로 그것을 실행할 때 바이러스 백신이 바이러스로 취급할 것이라는 것을 의미합니다.
