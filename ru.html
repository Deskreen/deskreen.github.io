Про Deskreen

Deskreen - это программа для компьютера, которая превращает любое устройство с веб-браузером в дополнительный экран для вашего компьютера через Wi-Fi. Deskreen можно использовать для отображения всего экрана компьютера на экране любого устройства, на котором есть веб-браузер. Также вы можете ограничить Deskreen выбором только одного окна приложения для совместного просмотра на другом устройстве - очень полезно для презентации. Лучшая функция Deskreen это использование любого устройства в качестве второго монитора, для этого Deskreen следует использовать с адаптером виртуального дисплея. Кроме того, без использования адаптера виртуального дисплея вы можете использовать телесуфлер с Deskreen на планшете или смартфоне, для этого Deskreen имеет режим переворота экрана, который отзеркаливает экран компьютера в веб-браузере планшета (переворачивает экран по горизонтали). Virtual Display Adapter . Also with no need for Virtual Display Adapter you can have a teleprompter with Deskreen using your tablet or smartphone, for that Deskreen has Flip Screen Mode that mirrors computer screen in web browser of your tablet (aka. flip screen horizontally).

Приложение можно использовать автономно, чтобы показывать экран людям в местах без доступа к интернету — только по Wi‑Fi. Мы успешно внедрили решения на базе Deskreen для партнёров для такого сценария.

Системные требования для Deskreen

Операционная система

Windows, Linux, MacOS

Доступная память на диске

210MB

ОЗУ

250 МБ в среднем для запуска приложения с одним сеансом демонстрации экрана. Каждый новый сеанс совместного использования экрана потребует в среднем до 100 МБ дополнительной памяти для работы без зависаний программы.

Процессор

Windows: любой современный двухъядерный процессор, более слабые процессоры могут иметь проблемы с производительностью с текущей версией в Windows. Мы собираемся исправить это в будущем.

MacOS: любой современный процессор

Linux: любой современный процессор, даже поддерживает 1 ядерную ARM систему, например Raspberry PI

Почему Deskreen?

Deskreen - это результат сочетания английских слов 'desktop' (рабочий стол) и 'screen' (экран).

"Все началось с того, что я хотел использовать свой iPad в качестве второго экрана для моего Macbook. Но я обнаружил, что мой IPad и Macbook уже устарели для использования встроенной функции MacOS под названием «Sidecar», и мне нужно купить новый iPad. У меня также есть ПК с Windows, и иногда мне нужно было использовать свой iPad в качестве второго экрана для него. Я пытался найти там БЕСПЛАТНЫЕ решения, но не смог найти ничего, что соответствовало бы моим требованиям. Кроме того, все методы и программное обеспечение, которые я нашел, были разными для Windows, Mac и Linux. Это было очень неудобно. Тогда у меня появилась идея создать Deskreen. Я решил сделать его бесплатным и с открытым исходным кодом, чтобы помочь людям решить эту проблему. Deskreen работает одинаково для Windows, MacOS и Linux."

В наши дни люди очень часто покупают новые устройства и могут иметь старый планшет, телефон или ноутбук, которым они перестали пользоваться. Если на вашем старом планшете, телефоне или ноутбуке по-прежнему работает браузер, Deskreen может помочь вам вдохнуть новую жизнь в вашего старого цифрового приятеля, и вы можете начать использовать его в качестве второго монитора для своего ноутбука."
