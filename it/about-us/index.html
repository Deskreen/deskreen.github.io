Informazioni su Deskreen

Deskreen è un applicativo desktop che trasforma qualsiasi dispositivo con un browser in uno schermo secondario per il tuo computer tramite WiFi. Deskreen può essere utilizzato per eseguire il mirroring dell'intero schermo del computer su qualsiasi dispositivo dotato di un browser. Inoltre puoi limitare Deskreen a selezionare solo una finestra da condividere - molto utile per scopi di presentazione. La caratteristica migliore di Deskreen è di poter utilizzare qualsiasi dispositivo come schermo secondario. Per avere una vera esperienza desktop estesa Deskreen dovrebbe essere usato con Adattatore di Visualizzazione Virtuale. Inoltre, senza bisogno di Adattatore di Visualizzazione Virtuale, puoi avere un Teleprompter usando il tuo tablet o smartphone, per questo Deskreen ha la modalità Capovolgi Schermo che specchia lo schermo del computer nel browser del tuo tablet (ovvero capovolge lo schermo orizzontalmente). Virtual Display Adapter . Also with no need for Virtual Display Adapter you can have a teleprompter with Deskreen using your tablet or smartphone, for that Deskreen has Flip Screen Mode that mirrors computer screen in web browser of your tablet (aka. flip screen horizontally).

Deskreen funziona anche completamente off‑grid: presenta lo schermo in luoghi senza Internet utilizzando solo una rete Wi‑Fi locale. Abbiamo realizzato con successo configurazioni offline basate su Deskreen per partner in questo scenario.

Requisiti di sistema per Deskreen

Sistema Operativo

Windows, Linux, MacOS

Spazio libero sul Disco

210MB

RAM

In media 250 MB per avviare una singola sessione di condivisione. Ogni nuova sessione avrà bisogno fino a circa 100 MB di memoria aggiuntiva per prestazioni fluide.

CPU

Windows: Qualsiasi CPU dual core moderna, CPU più deboli possono avere dei problemi di prestazioni con la versione corrente su Windows. La sistemeremo in futuro.

MacOS: Qualsiasi CPU moderna

Linux: Qualsiasi CPU moderna, supporta anche 1 core ARM, tipo Raspberry PI
