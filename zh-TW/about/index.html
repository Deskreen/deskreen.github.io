關於 Deskreen

Deskreen 是一款桌面應用程式，可以透過 WiFi 將任何帶有網路瀏覽器的裝置變成電腦的輔助螢幕。Deskreen 可用於將整個計算機顯示映象到任何具有 Web 瀏覽器的裝置螢幕上。您還可以限制 Deskreen 只選擇一個要共享的應用程式視窗檢視-這對於演示非常有用。Deskreen 的最大特點是可以使用任何裝置作為輔助螢幕。要獲得真正的擴充套件桌面體驗，Deskreen 應與虛擬顯示介面卡配合使用。此外，不需要虛擬顯示介面卡，您可以使用平板電腦或智慧手機擁有帶有 Deskreen 的提示器，因為 Deskreen 具有翻轉螢幕模式，可以在平板電腦的 Web 瀏覽器中映象計算機螢幕 (又名：水平翻轉螢幕)。 Virtual Display Adapter . Also with no need for Virtual Display Adapter you can have a teleprompter with Deskreen using your tablet or smartphone, for that Deskreen has Flip Screen Mode that mirrors computer screen in web browser of your tablet (aka. flip screen horizontally).

Deskreen also works completely off-grid: present your screen in places with no Internet, using only a local Wi-Fi network. We have successfully delivered Deskreen-based offline setups for partners in this scenario.

Deskreen 的系統要求

作業系統

Windows, Linux, MacOS

可用磁碟空間

210MB

記憶體

在一個螢幕共享會話中執行應用程式的平均大小為 250MB。每個新的螢幕共享會話平均需要高達 100MB 的額外記憶體才能流暢執行。

CPU

Windows: 任何現代雙核 CPU、較弱的 CPU 在 Windows 上的當前版本都可能出現效能問題。我們將來會把它修好的。

MacOS: 任何現代 CPU

Linux: 任何現代 CPU，甚至支援單核 ARM，例如 Raspberry Pi
