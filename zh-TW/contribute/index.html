為什麼您的貢獻對我們很重要

Deskreen 是為了幫助他人而打造的。Deskreen 永遠是完全免費的。此外，它是開源的，這意味著任何人都可以看到原始碼，以確保它不會對任何人造成任何傷害。Deskreen 完全是出於善意，並將由它來維持。

如果您出資，這將幫助我們保持動力，做錯誤修復和新增新功能，使 Deskreen 變得更好。

如果你是部落格作者，請考慮與你的朋友和訂閱者分享 Deskreen 的資訊。讓很多人知道 Deskreen 對我們來說很重要。

如果您是一名開發人員，歡迎您在我們的 Github 上發表意見。

我們要在 Deskreen 中完成的功能的待辦事項列表

程式碼簽名

錯誤修復

維護

儲存受信任的裝置 (自動連線)

將應用程式翻譯成其他語言

移除虛擬顯示介面解決方案 - (需要程式碼支援！)

支援更多平臺 (例如。樹莓派 (Raspberypi))

登入時自動啟動

透過系統托盤訪問

計算機的遠端控制

手寫筆支援 (用於繪圖)

對程式設計訪問的 CLI 支援

還有更多的...

您的貢獻將幫助我們更快地實現我們的目標。根據 Deskreen 社群的活躍程度，我們將在 Deskreen 的未來更新中實現上面列出的功能。
