Deskreen sekretesspolicy för skärmdelning och skärmvisning

Deskreen samlar inte in någon personlig information. Den lagrar bara inställningar lagrade på din dator, men dessa skickas aldrig utanför din dator.

IP -adresser kan lagras i loggar under en begränsad tid när du uppdaterar nyhetsbannern och letar efter uppdateringar, men det delas inte med någon.

Google Analytics-samtycke

Deskreen kör endast Google Analytics efter att du har gett tillåtelse, och du kan ändra beslutet när som helst.

Analysen förblir avstängd tills du uttryckligen tillåter spårning.

Du kan när som helst återkalla samtycket via integritetskontrollerna.

Behöver du stänga av spårning? Öppna kontrollerna nedan och välj Avvisa.

Tredjepartstjänster och annonsering

Denna webbplats använder tredjepartstjänster för att tillhandahålla analyser, visa annonser och förbättra användarupplevelsen. Dessa tjänster kan samla in information om ditt besök för att hjälpa oss förstå hur webbplatsen används och visa dig relevanta annonser.

Google Analytics

Vi använder Google Analytics för att förstå hur besökare interagerar med vår webbplats. Denna tjänst samlar in information som sidvisningar, tid som spenderas på sidor och navigationsmönster. All data anonymiseras och används endast för att förbättra vår webbplats. Du kan kontrollera Google Analytics-spårning via samtyckesbannern ovan.

Google AdSense

Denna webbplats visar annonser via Google AdSense. Google AdSense använder cookies och liknande teknologier för att visa dig personliga annonser baserade på dina intressen och surfbeteende. Du kan lära dig mer om hur Google använder data och hantera dina annonsinställningar på Googles sekretess- och villkorssida.

Ezoic

Vi samarbetar med Ezoic för att tillhandahålla annonstjänster och webbplatsoptimering. Ezoic kan samla in information om ditt surfbeteende för att leverera relevanta annonser och förbättra webbplatsens prestanda. Denna information kan inkludera din IP-adress, enhetsinformation och surfmönster.
